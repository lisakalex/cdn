def factorial(n):
    """Return the factorial of a given number."""
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)

# Example usage:
number = 5
print(f"The factorial of {number} is {factorial(number)}")

In this program:

  1. The factorial function is defined to calculate the factorial of a given number using recursion.
  2. The function checks if the input number n is 0 or 1, in which case it returns 1 (since 0! = 1 and 1! = 1).
  3. If n is greater than 1, the function recursively calls itself with n - 1 and multiplies the result by n.
  4. Finally, the program prints the factorial of the number 5.
